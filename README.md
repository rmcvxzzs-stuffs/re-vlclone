# re-vlclone

Complete clean room rewrite of vlclone (with better ui, leaning into the Netflix style)

re-vlclone introduces to new features, such as:

- Old QOL features from vlclone, now rewrote in a much better style
- Responsive UI
- Multiple files so it doesnt get dangling in one file
- Watch history (that syncs your ip)
- SQLite support (for watch history duh)
- 3 NEW themes (default, terminal-ish, and classic) (W.i.P)

re-vlclone also introduces:

## ReDB
Tiny helper for better-sqlite3. For use in re-vlclone only.
